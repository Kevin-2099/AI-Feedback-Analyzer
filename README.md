# 🚀 AI Feedback Analyzer

Una aplicación para analizar automáticamente el feedback de usuarios. Detecta idioma, clasifica sentimiento, genera resúmenes, identifica problemas comunes y ofrece recomendaciones para mejorar productos o servicios.

## Características

- Detección de idioma (español, inglés y otros)
- Análisis de sentimiento con emojis
- Resúmenes automáticos de comentarios
- Identificación de problemas frecuentes
- Recomendaciones automáticas basadas en los problemas detectados
- Interfaz sencilla con Gradio

## Instalación

1. Clona este repositorio:

git clone https://github.com/Kevin-2099/AI-Feedback-Analyzer.git

cd ai-feedback-analyzer

Crea un entorno virtual e instala dependencias:

python -m venv venv

source venv/bin/activate  # en Windows: venv\Scripts\activate

pip install -r requirements.txt

Ejecuta la app:

python app.py

## Uso

Abre la interfaz de Gradio en tu navegador.

Pega el feedback de usuarios o comentarios.

Observa los resultados en cinco secciones: idioma, sentimiento, resumen, problemas detectados y recomendaciones.

## Tecnologías

Gradio
 – interfaz web interactiva

Transformers
 – análisis de sentimiento y resúmenes

langdetect
 – detección de idioma
